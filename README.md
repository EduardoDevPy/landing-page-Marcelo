# MB Advocacia — Landing Page

Landing page profissional para o escritório **Marcelo Bueno Advocacia**, especializado em direito empresarial, trabalhista e cível.

## Tecnologias

- [React](https://react.dev/) + [TypeScript](https://www.typescriptlang.org/)
- [Vite](https://vitejs.dev/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Shadcn UI](https://ui.shadcn.com/) (Accordion, Carousel)
- [Lucide React](https://lucide.dev/)
- Fontes: **Cinzel**, Playfair Display, Inter (Google Fonts)

## Seções

- **Hero** — Headline principal com CTA para WhatsApp
- **A Realidade** — Problemas jurídicos comuns que empresários enfrentam
- **Sobre** — Apresentação do escritório
- **Números de Impacto** — Métricas com animação de contagem
- **Atuação** — Áreas de atuação (Empresarial, Trabalhista, Cível) com cards
- **Benefícios** — Carrossel com os benefícios da assessoria jurídica
- **Como Funciona** — Metodologia de trabalho
- **Dúvidas Frequentes** — FAQ com accordion interativo
- **CTA Final** — Chamada para ação
- **Footer**

## Como rodar localmente

```bash
npm install
npm run dev
```

## Build para produção

```bash
npm run build
```
